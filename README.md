## 🔐 Projects

> Three builds. Each one solves a real security problem.

---

### 🔒 Host-Based Intrusion Detection System (HIDS)
*"Detect threats before they become breaches."*

Built a Python-based HIDS that monitors file integrity, detects brute-force attacks, flags suspicious processes, and catches unauthorized network activity — all in real time.

- ✅ SHA-256 file integrity monitoring
- ✅ 100% test case success rate
- ✅ Scans complete in under **6.4 seconds**
- ✅ Live dashboard — LOW / HIGH / CRITICAL severity (Jupyter + Plotly)

---

### 🔍 PortSentinel — TCP Port & Service Scanner
*"Know what's exposed before the attacker does."*

Python-based TCP port scanner covering ports 1–1024. Detects active services (HTTP, FTP, SSH) via real socket communication — lightweight, fast, precise.

---

### 🎣 PhishTrace — Secure URL & Email Analyzer
*"Catch phishing before it catches you."*

Rule-based phishing detection engine using 10+ heuristic indicators. Classifies threats as **Low / Medium / High** risk with feature-based scoring.
